Auto-Scroll Helper

Since iOS Safari restricts JavaScript bookmarklets, use this page to activate auto-scrolling:

  1. Tap the button below to copy the auto-scroll code
  2. Go to the article you want to read
  3. Tap in the Safari address bar
  4. Paste the code and tap Go
